# 📝 To-Do List Application (Python)

A simple and user-friendly To-Do List application built using **Python and Tkinter**. This project helps users efficiently manage their daily tasks with a clean graphical interface.

## 🚀 Features

* Add new tasks
* Delete existing tasks
* Mark tasks as completed
* Persistent task storage using JSON
* Simple and intuitive GUI

## 🛠️ Technologies Used

* Python
* Tkinter (GUI)
* JSON (Data storage)


## 📂 Project Structure

```
├── tasks.json        # Stores tasks
├── todo_app.py       # Main application file
```

## ▶️ How to Run

1. Clone the repository
2. Navigate to the project folder
3. Run the Python file:

```bash
python todo_app.py
```
